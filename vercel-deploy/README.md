# Tripletex Partner Cloud v0.4

## Deploy to Vercel (30 seconds)

1. Go to https://vercel.com/new
2. Click "Deploy from ZIP"  — or drag this folder
3. Done! You get a public URL like `tripletex-partner-cloud.vercel.app`

## Deploy to Netlify (30 seconds)

1. Go to https://app.netlify.com/drop
2. Drag the `index.html` file onto the page
3. Done! Instant public URL

## Run locally

Just double-click `index.html` — no server needed.
